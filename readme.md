# Calhappy Project

Questo è il progetto Calhappy. Segui questi passaggi per attivare l'ambiente e avviare il codice.

## 1. Creare e attivare l'ambiente virtuale

Prima di tutto, crea l'ambiente virtuale. Una volta creato, attivalo.

Quando l'ambiente è attivo, dovresti vedere `(calhappy)` nel prompt.
attivazione ambiente con

source calhappy/bin/activate


## 2. Installare le librerie necessarie

Con l'ambiente attivo, installa le librerie necessarie per il progetto.

Puoi aggiungere altre librerie che potrebbero essere necessarie.
pip install -r requirements.txt


## 3. Eseguire il codice

Per eseguire il codice, avvia lo script principale `calhappy.py`. 

python3 calhappy.py
oppure
./run.sh

Questo avvierà lo script che gestisce l'invio degli inviti e la generazione delle frasi ispirazionali.
